Introduction

What is HTML?

HTML, or Hypertext Markup Language, is used to tell browsers how to render the information in websites that a user visits. For Example:

<h1>Hello World</h1>

The above example is raw code and not what a user should see. The user will see, when properly rendered, the following:

Hello World

In the above example the code was rendered correctly. Note how the font is larger and bolded. This comes from the tag itself telling the browser how to behave.

Basics
<span>I am an example of a typical <strong>HTML</strong> element</span>

These are the standard parts of an HTML element.

  1. Opening Tag - The name of the element enclosed in angled brackets. This tells the browser where the element begins or where to start an affect, such as the <strong> element changing text to be bold.
  2. Content - The meat and potatoes. This is the actual information that is presented to the user, such as the text you are reading right now. However, content is not limited to just text. It can be an image or video, or even a game.
  3. Closing Tag - This is the tag at the end of the element. It tells the browser where the element stops or an affect end, such as only the word "HTML" in the above is bold. Without a closing tag, the browser will assume that all that follows that opening tag is part of that element. So watch out for that. Some tags are self-closing, meaning they don't need a closing tag.
  4. Element - This is what the opening tag, content and closing tag are collectively known as.
Nesting, Classes, and ID's

Nesting - The act of adding an element to another element, also used to refer to adding content. All of the element being nested must be inside the opening and closing tags of the parent element. Like so:

<div><span>I am an example</span></div>

In the above example, the span element was nested in the div element. In this case the span element is known as the child of the div element, which is the parent element.

Classes - A class is a name given to an element, usually used to style that element. A class can be given to multiple elements. For example:

  • <div class="example">I am example 1</div>
  • <div class="example">I am example 2</div>
  • <div class="example sample">I am example 3</div>

Classes are denoted with a period at the front. In the above examples, all three elements have the class .example, and only one has the class .sample. So if .example is targeted for styling, all three would have that styling applied. However if .sample is targeted then only example 3 would be affected.

ID's - An id, written as id="example" is similar to classes. However there are 2 main differences. An ID must be unique to a single element, for example, if id="sample" is used for an element then it can't be used again. The second difference is that instead of a period to denote an ID, a pound sign(#) is used.

Block and Inline Elements

Block - Elements in this family are always started on a new line. They take up the full width available, such as the entire parent element in which they are located. Two of the most common block elements are the p and the div elements. As an example:

I am a div!

Inline - Inline elements are elements that do NOT start a new line. They also only take up as much width as necessary in order to hold their content. An inline element can be nested inside a block level element, however an inline element can not hold a block level element. As an example:

I am before an inline element; I am an inline element! And I am after the inline element!

Notice how the inline element, in this case a span, is the only part of the element with a red background. The rest of the element is a p element.

CSS

What is CSS?

CSS, or Cascading Style Sheets, is a language used to describe how an HTML element is displayed. It can control multiple web pages at once. It removed the clutter of trying to style HTML from the inside. CSS controls many things in an HTML document, from font size, style and color, to adding animation to elements. The page you are reading right now was extensively styled with CSS.

p { color: green; }

The above is a standard example of CSS styling. In this case the styling would change the font color to green in any p element present on the page.
